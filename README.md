# pyproj_maker
template for my python projects

## Usage:

To create a project sceleton call:

    python pyproj_make.py --name <lib_name> --targetfolder <path/to/lib>

to check the result:

    (cd <path/to/lib>/<lib_name>/tests && sh test_install.sh)

## Requirements:

   * For Linux (sh)
   * Python>=3.5
